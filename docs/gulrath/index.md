# Gul'Rath

The northern continent where the original campaign began. Home to yuan-ti empires, dwarven holds, and the mysterious Magisterium.

## Power Rankings

### Faction Power (S-D Scale)
Measures combined military, economic, and diplomatic might.

| Tier | Description |
|------|-------------|
| **S** | Continental superpower |
| **A** | Major regional power |
| **B** | Significant regional influence |
| **C** | Minor regional player |
| **D** | Local or declining power |

### Heroic Rankings (S+ to D)
Individual power based on character level and renown.

| Tier | Level | Description |
|------|-------|-------------|
| **S+** | 21+ | Godlike beings |
| **S** | 17-20 | Master heroes of legendary renown |
| **A** | 14-16 | Realm heroes of great renown |
| **B** | 10-13 | Regional heroes of some renown |
| **C** | 6-9 | Local heroes of great renown |
| **D** | 1-5 | Local heroes of little renown |
