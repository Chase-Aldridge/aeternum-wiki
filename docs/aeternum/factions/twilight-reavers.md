# THE TWILIGHT REAVERS

| | |
|---|---|
| **Leader** | Guildmaster Oreon, Lu Zhong, Gromwinston, Ryuzen |
| **Alignment** | Neutral |

## Description

MERIA's elite mercenary force, headed by four warlords:
- GUILDMASTER OREON
- LU ZHONG THE TIGER
- GROMWINSTON THE GARRULOUS
- RYUZEN THE DRUNKEN PHOENIX

They amassed a large military and trade fleet and set sail north in 1114AE into the uncharted seas with the blessing of THE VOICE using old maps from the dawn era.
