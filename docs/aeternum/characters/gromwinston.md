# GROMWINSTON THE GARRULOUS

**Title:** Warlord of the Twilight Reavers
**Race:** Unknown

## Description

GROMWINSTON THE GARRULOUS is one of the new warlords heading MERIA's elite mercenary force known as THE TWILIGHT REAVERS.

Known for his verbose nature, he set sail north in 1114AE with the other warlords into uncharted waters.
